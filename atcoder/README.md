- Tasks
  - 問題の名前
- Time
  - XX:XX : 解くのにかかった時間がXX:XXという意味
  - ~XX:XX : 前の問題を解くのにかかった時間＋その問題を解くのにかかった時間がXX:XXという意味
- Notes
  - 発見や失敗
- Status
  - o : 自力で問題なく解けた
  - r : どうにかして解けたが、復習する必要を感じている
  - x : 挑戦したが解けなかった。解説を見ても理解できなかった。
